## 🚀 Deployment

This is a single-page HTML application with embedded CSS and JavaScript — zero dependencies.

### Option 1: GitHub Pages
```bash
# Push to main branch — GitHub Pages auto-deploys
git push origin main
# Then enable in repo Settings → Pages → Deploy from main branch
```

### Option 2: Netlify (Recommended)
```bash
# Drag the index.html file to https://app.netlify.com/drop
# Or connect GitHub repo — Netlify auto-deploys
```

### Option 3: Any Web Host
Just upload `index.html` to any static web host — it's self-contained.
